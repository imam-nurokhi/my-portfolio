# Muhammad Imam Nurokhi – Portfolio

A professional portfolio website built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Forms**: React Hook Form + Zod validation
- **Fonts**: Playfair Display + Inter (Google Fonts)
- **Language**: TypeScript

## ✨ Features

- 🌑 **Dark Glassmorphism theme** with Electric Blue & Soft Gold accents
- ✏️ **Typing animation** cycling through professional roles
- 🌌 **Particle system** background animation with network connections
- 📜 **Scroll-triggered** experience timeline with SVG path drawing
- 🗃️ **Bento Grid** project showcase with modal overlays
- 📬 **Contact form** with React Hook Form + Zod validation & micro-interactions
- 💡 **System Status** footer with live Jakarta time
- 📱 **Fully responsive** across all devices
- �� **SEO optimized** with Next.js metadata API + Open Graph

## 🏗️ Sections

1. **Hero** – Animated headline, typing roles, particle background, stats
2. **Expertise** – Core domain cards + technology arsenal grid
3. **Experience** – Animated vertical timeline of work history
4. **Projects** – Bento grid with clickable project modals
5. **Contact** – Form with validation + LinkedIn integration
6. **Footer** – System status component + digital signature animation

```bash
npm install
npm run dev
```
